# 标准化销售培训AI系统
基于Python+LangChain+分层RAG搭建的通用销售实训原型，面向各行业销售人员标准化话术培训，采用固定两段式LLM链式流程，保证考核评判标准统一可控。

## 技术栈
Python、LangChain、RAG检索增强、LLM API、Chroma向量数据库

## 四大核心模块架构
1. **决策树流程调度引擎**
使用结构化JSON存储考核分支规则，管控完整训练对话流程，限制大模型自由发散闲聊，训练流程标准化可追溯。
2. **参数化动态提问生成模块**
离线+线上双隔离RAG架构：离线完成销售素材清洗、切片、向量化入库；线上仅本地拼接变量生成客户对话，训练全程不调用LLM，控制接口成本。
3. **轻量化LLM流程判定模块**
学员每轮作答后执行通关达标校验，后台静默记录沟通短板，不中途弹窗干预，保证测评结果客观真实。
4. **两段式串行Chain评分复盘模块**
训练结束串行执行两轮LLM调用：打分环节输出结构化量化得分；复盘环节拼接对话记录、打分结果、RAG检索标杆话术，生成完整个性化改进报告。

## 后续开发计划

当前版本优先完成核心训练闭环：决策树训练、学员作答、LLM 节点判定、短板记录、RAG 复盘与 AI Coach 报告生成。后续将围绕题库扩展、能力画像和企业级应用能力继续迭代。

###  能力画像系统
- 记录学员每次训练中的节点表现、短板标签与评分结果
- 按能力维度生成长期成长记录
- 支持查看学员在共情沟通、异议处理、产品价值表达、合规沟通等维度的变化趋势

###  离线 RAG 题库扩充流水线
- 支持导入企业产品资料、销售案例、标准话术和合规文档
- 使用 LangChain 对资料进行切分、向量化和元数据标注
- 根据客户类型、考核能力和业务场景自动生成候选训练题目
- 将生成结果转化为结构化模板和变量池，供决策树训练引擎调用

###  多场景训练扩展
- 增加价格异议、竞品比较、需求模糊、强势质疑等更多客户类型
- 支持不同难度等级的训练节点
- 根据学员短板自动推荐下一轮训练场景

###  企业级训练平台
- 支持多学员训练记录管理
- 支持培训管理员配置题库、评分 Rubric 和知识库资料
- 支持训练结果导出、团队能力分析和标准化培训评估
